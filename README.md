# creditcard_fraud_detection
這個專案的目標是開發一個機器學習模型，以有效處理信用卡交易數據。最終
目標是使用“Class”特徵作為反應變數，Class值為1表示詐騙交易，Class值為0表
示正常交易，以精確識別和檢測信用卡詐騙，以在檢測到異常模式時及時通知相關
人員，從而保護客戶免受未經授權的費用
